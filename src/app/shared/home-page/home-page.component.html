VegLab

Saisie, traitement et analyse de données
pour les végétations et les habitats

Présentation

VegLab est un projet porté par Tela Botanica visant à faciliter l'accès et le partage de données de végétations et d'habitats naturels. Il s'agit d'une plateforme (application et base de données) libre d'utilisation vous permettant de gérer vos données depuis leur saisie jusqu'à leur analyse. Le projet est actuellement en cours de développement et toutes les fonctionnalités ne sont pas encore accessibles.

Espaces de travail

Un espace de travail vous donne accès à toutes les fonctionnalités de base de l'application. Vous disposez d'une interface adaptée, de modules spécifiques à votre activité ainsi que de l'accès aux données qui vous sont les plus utiles.

Pour commencer, choisissez un espace de travail.
