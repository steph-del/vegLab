Inclure les relevés sans métadonnées

Lorsque vous utilisez un ou plusieurs filtres sur les métadonnées, vous pouvez choisir d'inclure ou non les relevés pour lesquels ces métadonnées n'ont pas été renseignées.


Exemple : vous souhaitez filtrer les relevés avec :

  • un recouvrement compris entre 0 et 50%
  • ET une aire inférieure à 10 mètres carrés

Dans ce cas, l'option vous permet de choisir si vous souhaitez inclure les relevés pour lesquels ni le recouvrement ni l'aire ne sont renseignés :

Option activée

Les relevés ont

  • un recouvrement compris entre 0 et 50%
  • ET une aire inférieure à 10 mètres carrés

OU

N'ont pas de données renseignées pour ces l'une ou l'autre de ces métadonnées

Option désactivée

Les relevés ont

  • un recouvrement compris entre 0 et 50%
  • ET une aire inférieure à 10 mètres carrés

 

Les relevés ne présentants aucune donnée de recouvrement ni d'aire sont ignorés.
